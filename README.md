# Dating App

Tutorial to get acquainted with the latest versions of my favorite backend & frontend framework combo:

* .NET 10
* Angular 21

## Tutorial Info

* [Udemy course page](https://www.udemy.com/course/build-an-app-with-aspnet-core-and-angular-from-scratch/?couponCode=DAANGUPDATE16)

Check out this Udemy course, I'm working through it for the 3rd time in 8+(?) yrs. This instructor updates it completely when it gets a bit dated--each time has introduced new concepts. i.e., for Angular--seems like the first time only promises were used to return data, the second time was an intro to observables with subscriptions, and this time I'm learning the use cases for signals.
